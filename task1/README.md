# Task 1

## Usage

```sh
cargo run <file>
```

Example:
```sh
cargo run example.txt
```

Output:
```
File analysis results:
- Lines: 10
- Words: 50
- Characters: 200
```

### Help

```sh
cargo run help
```

Displays usage information about the program.

## Requirements
- Rust 1.65+
